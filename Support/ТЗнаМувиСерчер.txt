Тестовое задание на React Native/Kotlin:

Требуется написать приложение для поиска фильмов используя IMDB Api.

Для начала вам потребуется создать ключ для работы с Api, он создается по этой ссылке http://www.omdbapi.com/apikey.aspx

Необходимо реализовать два экрана.


Первый экран:
Сверху заголовок “Movie searcher”
Ниже: строка поиска с кнопкой “Search” 
При нажатии на кнопку Search будет переход на экран, где будет выводиться результат поиска.
Запрос на поиск: https://www.omdbapi.com/?s=${searchValue}&apikey=${yourKey}

Также требуется добавить кнопку для фильтрации, фильтровать можно по двум параметрам (фильтры на год и тип фильма)
https://www.omdbapi.com/?y=${year}&apikey=${yourKey}
https://www.omdbapi.com/?type=${type}&apikey=${yourKey}
Типы фильмов: movie, series, episode
Должна быть кнопка сброса фильтров до начальных значений

Далее должен выводится список рекомендованных фильмов или сериалов, нужно делать запрос со статическим значением параметра “s”, значение выбираете сами.

Второй экран:
Отображается результат поиска по фильтрам и введенному значению в поле поиска
Должен быть возврат на предыдущий экран.
